Obsidian News Desk

Automated News Video Production Pipeline

Welcome to Obsidian News Desk

This wizard will guide you through the installation process. The setup takes approximately 5-10 minutes and will configure everything you need to start producing professional news videos.

System Requirements

○
At least 10GB free disk space
○
Minimum 8GB RAM recommended
○
Node.js 18+ (bundled with installer)

What will be installed:

  • Next.js web application
  • Docker containers (PostgreSQL and Redis)
  • BullMQ job queue system
  • Remotion video rendering engine

Docker Desktop

Obsidian News Desk requires Docker Desktop to run PostgreSQL and Redis containers. We'll check if Docker is installed and running.

ℹ Click "Check Docker" to begin detection.

Note: If Docker Desktop is installed for the first time, Windows may require a restart. The installer will resume after reboot.

Storage Location

Choose where to store generated videos, images, and avatars. Make sure you have at least 10GB of free space.

The following directories will be created:

  • images/ - Scene background images from Whisk API
  • avatars/ - HeyGen avatar MP4 files (30-60 MB each)
  • videos/ - Final rendered broadcast videos
