Whisk Token Setup (Automated)

The included Chrome extension automatically captures and refreshes your Whisk token every 50 minutes.

✨ Recommended Method: Chrome Extension (Automatic)
No manual token copying needed! The extension handles everything.

Install Chrome Extension (2 minutes)

1

Open Chrome Extensions

Navigate to chrome://extensions/ in Chrome or Edge

2

Enable Developer Mode

Toggle "Developer mode" in the top-right corner

3

Load Extension

Click "Load unpacked" → Select folder:
C:\Program Files\Obsidian News Desk\resources\chrome-extension\

4

Visit Whisk & Generate Test Image

Go to labs.google.com/fx/tools/whisk
Generate any test image (the extension captures the token automatically)

5

Verify Token Captured

Click the extension icon (green key) in Chrome toolbar
Should show "Token Active" with green indicator

🔄 Auto-Refresh: The extension automatically refreshes your token every 50 minutes. No manual action needed!
Alternative: Manual Token Setup (Not Recommended)

If you prefer manual setup:

  1. Open labs.google.com/whisk
  2. Press F12 → Network tab
  3. Generate a test image
  4. Find request to "generateImage"
  5. Copy Authorization header: "Bearer XXX..."
  6. Paste in app Settings → Whisk Token

Note: Manual tokens expire every hour and need to be updated.

HeyGen Avatar Workflow

Avatar generation is a manual step - the system will launch HeyGen for you when ready.

1

Wait for Review Assets State

After images are generated, the job will pause for human review

2

Click "LAUNCH HEYGEN BROWSER"

The system will open HeyGen.com in your browser

3

Generate Avatar

Paste the avatar script, select your avatar, and generate the video

4

Download .MP4 File

Download the generated avatar video (usually 30-60 MB)

5

Optimize if Needed

If file is >10MB, use optimize-avatar.sh script to reduce size

6

Upload via Storyboard Editor

Drag and drop your avatar file, then click "COMPILE & RENDER"

💡 Optimization Tip: Large avatar files can cause Remotion timeout. Always optimize files >10MB for best performance.
